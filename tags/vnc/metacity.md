# Metacity

## Start Metacity (normal mode) in VNC

```bash
metacity &
```

## Start Metacity and replace the current window manager

If another window manager is running (e.g., Openbox, Mutter):

```bash
metacity --replace &
```

##  Start Metacity without session management

```bash
metacity --sm-disable &
metacity --sm-disable --replace &
```
