# Git List

## Git List files in the Git Index

### List Tracked Files

```bash
git ls-files
```

### List Files Matching a Pattern

```bash
git ls-files "*.txt"
```

### List Files in the Index

```bash
git ls-files --stage
```

### List Modified Files

```bash
git ls-files --modified
```

### List Deleted Files

```bash
git ls-files --deleted
```
