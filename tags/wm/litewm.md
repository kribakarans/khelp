# Light Weight Window Managers

| Use Case             | Recommended WM |
| -------------------- | -------------- |
| Ubuntu-style minimal | **Metacity**   |
| Light + configurable | **Openbox**    |
| Ultra-light floating | **Fluxbox**    |
| Built-in panel       | **IceWM**      |
| Keyboard power user  | **i3**         |
| Embedded / kiosk     | **Matchbox**   |
